Les displays

display: block

Cette div est un élément en Block. Il occupe toute la largeur de lécran

Exemple de balises de type 'block' :

  • div
  • p
  • h1,h2,h3,h4
  • li
  • main
  • ul

Display : inline

Les éléments en inline ne forcent pas le retour à la ligne, il prend simplement la place dont il a besoin

Attention les éléments de type inline ne sont pas commme les types block, leur hauteur et largeur ne sont délimités que par leur contenu

Exemple de balises e type inline

  • balise a (liens)
  • img
  • input
  • video
  • span
  • label

Display : Inline-block

Inline-block: c'est un type inline qui conserve les propriétés du type block, il peut donc être modifié en hauteur et largeur comme n'import quel type block

Display: flex

Flex: Les éléments dans le conteneur s'alignent en ligne et se répartissent l'espace disponible

Display: Grid

Grid : avec grid, on peut définir une grille avec des lignes et des colonnes

Display: None

None: le display none cache tout simplement l'élément.
